## Çalıştırmak İçin

- `npm install -g electron`
- Proje dizininde;
- `npm install`
- `electron .`

# &nbsp;

## Build Almak İçin

- `npm install -g electron-packager`
- Proje dizininde;
- `electron-packager ./`
